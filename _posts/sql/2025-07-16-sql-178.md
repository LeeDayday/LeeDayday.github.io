[문제 링크](https://leetcode.com/problems/rank-scores/)

### 정답
```sql
SELECT
    score,
    DENSE_RANK() OVER (ORDER BY score DESC) as 'rank'
FROM Scores
ORDER BY score DESC;
```

#### ✅ Point
- column name을 `rank`로 하면 에러 발생 (rank라는 함수가 존재하므로 syntax error 가 발생)
    - 따라서 `''`로 감싸줌
