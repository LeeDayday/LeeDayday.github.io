### 정답
```sql
-- ANIMAL_INS: ANIMAL_ID, ANIMAL_TYPE, DATETIME, INTAKE_CONDITION, NAME, SEX_UPON_INTAKE
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NULL
ORDER BY 1;
```

#### ✅ Point
- WHERE문을 통해 NAME 이 NULL 인 경우만 필터링
