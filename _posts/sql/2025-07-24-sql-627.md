[문제 링크](https://leetcode.com/problems/tree-node/)

### 정답
```sql
UPDATE Salary
SET sex = 
CASE 
    WHEN sex = 'f' THEN 'm'
    ELSE 'f'
END;
```

#### ✅ Point
- 성별을 toggle 하는 update 문
- CASE WHEN 은 SELECT, INSERT, DELETE, UPDATE 등에서 동일한 방법으로 사용 가능
