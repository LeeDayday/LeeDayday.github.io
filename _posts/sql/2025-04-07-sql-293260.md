### 정답
```sql
-- FISH_INFO: ID, FISH_TYPE, LENGTH, TIME
SELECT 
    COUNT(IDㅁ) AS FISH_COUNT,
    MONTH(TIME) AS MONTH 
FROM FISH_INFO
GROUP BY MONTH(TIME)
ORDER BY 2;
```

#### ✅ Point
- 월별 집계: MONTH(TIME)을 기준으로 GROUP BY 하여 각 월에 잡힌 물고기 수를 집계
- 카운트 대상: COUNT(ID)를 통해 각 월별로 잡힌 물고기의 총 개수 계산
- 정렬 기준: ORDER BY 2를 통해 MONTH 순으로 오름차순 정렬
